# PocketCore
### Version 1.0

## How to Setup!
1) Go src/main.
2) Go Down Until You find the Messages That Start With $p->sendMessage("");.
3) Edit the text in $p->sendMessage("EDITABLE TEXT IS HERE");
4) Save add to server & reload
5) Complete!
